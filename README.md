## 🚀 Features

- **Recipe Article Layout**
  - Structured recipe page with title, subtitle and descriptive content
  - Clean visual hierarchy for an easy reading experience

- **Hero Recipe Section**
  - Main heading for the recipe
  - Introductory paragraph
  - Large featured image
  - Photo credits section

- **Featured Link Section**
  - Highlighted area for related recipe content
  - External reference link for further reading

- **Ingredients Section**
  - Separate subsections for:
    - focaccia dough
    - topping ingredients
  - Clear bullet-list organization

- **Method Section**
  - Ordered recipe instructions
  - Focus on readability and semantic HTML structure

- **Clean Static Styling**
  - Custom CSS for spacing, typography and layout
  - Centered content container
  - Highlighted subtitle banner
  - Styled image section with rounded corners
  - Section dividers for better content separation

- **Educational Context**
  - Built as an early HTML/CSS exercise to practice static webpage creation and content structuring
